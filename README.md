# 居家考核说明（统一说明）

本考核为 **居家完成型实战考核**，候选人可自由使用：

* 各类搜索引擎、技术文档
* GitHub / StackOverflow
* **各类 AI 编程或大模型工具（如 ChatGPT、Claude、Copilot 等）**

我们**鼓励**合理使用 AI 工具。

> 本考核的核心目的不是测试“记忆力”或“纯手写能力”，而是评估：
>
> * 学习能力
> * 问题拆解能力
> * 工程与产品思维
> * 对 AI 能力边界的理解与运用

---

## 考核形式（统一）

* 我们会提供一个 GitHub 仓库地址
* 你需要 **Fork 仓库 → 在自己仓库中完成 → 提交 Pull Request**
* 允许你修改或新增任何文件
* 请确保：

  * 能通过 README 理解你做了什么
  * PR 描述清晰

### 时间要求

* **建议投入时间：6～10 小时**
* **最长完成时间：1.5 天**


* 不要求功能全部完成，请合理取舍
